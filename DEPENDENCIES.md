# Dependências Adicionais Necessárias

Para que o projeto funcione completamente, você precisa instalar as seguintes dependências:

```bash
npm install tailwindcss-animate react-day-picker
```

## Dependências Opcionais (para funcionalidades futuras)

- `jspdf` - Para exportação de PDF
- `ics` - Para exportação de iCal

## Notas

- O componente `Calendar` em `components/ui/calendar.tsx` usa `react-day-picker` que precisa ser instalado
- O `tailwind.config.ts` referencia `tailwindcss-animate` que precisa ser instalado
